LiveMedBench

A Live Medical Benchmark for Evaluating Large Language Models

LiveMedBench is designed to measure not only overall medical quality, but also robustness over time under a rubric-based evaluation framework using real-world, live-update medical data.

Data Collection

Continuous mining of bilingual clinical data from verified online medical communities.

Multi-Agent Curation

A hierarchical framework (Screener, Validator, Controller) structures and validates data against medical guidelines.

3

Rubric Generation

Automated generation of case-specific evaluation rubrics with bipolar criteria and weighted axes.

4

Model Evaluation

Objective evaluation of LLMs using the generated rubrics via a rubric-based grader.

Human QA

Rigorous human quality assurance to ensure clinical alignment.
